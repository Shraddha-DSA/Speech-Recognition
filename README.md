# 🎧 Speech Emotion Recognition using Librosa + CNN

This project predicts human emotions from speech audio using **Librosa** for feature extraction and **CNN** for classification.

---

## 🚀 Features
- Mel Spectrogram extraction using Librosa
- CNN-based deep learning model
- Emotion prediction from audio
- RAVDESS dataset support

---

## Emotions Supported
- Happy
- Sad
- Angry
- Neutral
- angry
- fearful
- disgust
- surprised
---
## Tech Stack
- Python
- Librosa
- TensorFlow / Keras
- NumPy
- Scikit-learn

---
## Dataset

RAVDESS emotional speech dataset
